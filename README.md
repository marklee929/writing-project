# Writing Project

AI와 함께 소설을 쓰고, 기록하고, 실험하는 공간.

이 저장소는 인간의 상상력과 AI의 보조를 결합하여
장편 소설, 세계관, 설정 문서, 초안 및 작업 로그를
지속적으로 축적하기 위한 개인 프로젝트이다.

## Current Projects

- NOSE CUT
- Ryla
- Artifitual Revolutionist

## Philosophy

AI는 저자가 아니다.

AI는 아이디어 정리, 구조 검토, 문서화,
초안 작성 보조 역할을 수행하며,
최종 방향과 판단은 인간 저자가 결정한다.

이 저장소의 목적은
'좋은 작품을 만드는 것'만이 아니라,
인간과 AI가 함께 이야기를 만들어가는 과정을 기록하는 것이다.
